# PDF Pages（單頁資料編輯）功能介紹

## 概述

「PDF Pages」模組是 QS 小工具的核心頁面瀏覽與資料編輯介面。使用者可以逐頁查看已上傳的 PDF 檔案，為每頁定義欄位（Column），輸入或編輯文字資料，並能透過模板（Template）快速套用固定欄位佈局，配合 OCR 自動辨識文字。

## 功能

### 頁面瀏覽
- 以檔名及頁碼顯示所有已載入頁面
- 透過下拉選單或前後翻頁按鈕快速切換
- 選定的頁面會同步顯示在右側 PDF 檢視器中

### 欄位管理
- 自訂欄位名稱（如「項目」、「數量」、「單價」等）
- 新增與刪除欄位，靈活配合不同表格結構
- 每個欄位對應一個可編輯的文字輸入框

### 逐格資料編輯
- 在「Column | Text」表格中，點擊任何欄位即可輸入或修改文字
- 適合人工校對和微調 OCR 辨識結果

### 模板系統
- 從已套用的框選位置建立新模板
- 更新現有模板以反映最新的框選佈局
- 套用模板至其他頁面，實現一致的資料欄位定義

### OCR 辨識
- 點擊「🔍 Recognize」按鈕，對當前頁面進行文字辨識
- 辨識結果自動填入相應欄位
- 辨識消耗配額（可在 HomePanel 查看用量）

## 使用流程

### 基本操作

1. **上傳 PDF**：在工具列點擊「📂 開啟 PDF」上傳一或多個 PDF 檔案
2. **選擇頁面**：在頁面下拉選單中選擇要查看的頁面，或用 ◀ / ▶ 按鈕翻頁
3. **定義欄位**：在底部輸入框輸入欄位名稱，按 Enter 或點「Add」新增欄位
4. **手動輸入**：在表格中的 Text 欄位直接鍵入文字

### 搭配模板

1. 在 PDF 檢視器上拖曳框選出各欄位的位置區域
2. 輸入模板名稱，點擊「➕ New」儲存為新模板
3. 翻到其他相同格式的頁面，從模板下拉選單選擇並套用
4. 點擊「🔍 Recognize」辨識文字，自動填入欄位
5. 檢查並修正辨識結果

### 小技巧

- 模板套用後，右側 PDF 檢視器上會顯示對應的彩色框選區域
- 同一模板可跨檔案使用，只要頁面版面格式相同即可
- 辨識前先確認框選位置正確，以提高 OCR 準確率
