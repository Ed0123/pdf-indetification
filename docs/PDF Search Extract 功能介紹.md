# PDF 搜尋與擷取 功能介紹

## 概述

「PDF 搜尋與擷取」模組讓使用者針對多頁 PDF 檔案，透過自訂關鍵字規則搜尋頁面內容，並將符合條件的頁面擷取到新的 PDF 檔案中下載。支援萬用字元搜尋（`*` 和 `?`），所有處理皆在瀏覽器本機完成。

## 功能

### 關鍵字規則管理
- 可新增多組搜尋規則，每組規則包含關鍵字、排除名單、標記顏色與透明度
- 規則可個別啟用/停用（勾選框控制）
- 每組規則的關鍵字以逗號分隔，支援同時搜尋多個關鍵字
- 每組規則可設定自身的「排除名單」，包含排除關鍵字的頁面不會被該規則選中

### 萬用字元搜尋
- `*` — 代表任意數量的任意字元（包括零個字元）
  - 範例：`Section *` 可匹配 `Section 1`、`Section A.2`、`Section Notes` 等
- `?` — 代表恰好一個任意字元
  - 範例：`Item ?` 可匹配 `Item A`、`Item 1`，但不匹配 `Item 10`
- 可與一般文字混用，例如 `P?-*-Rev?` 可匹配 `P1-Foundation-RevA`

### 全域白名單 / 黑名單
- **全域白名單**：頁面必須同時包含所有白名單中的關鍵字，才有可能被選中
- **全域黑名單**：頁面只要包含任一黑名單關鍵字，即立刻排除
- 白名單和黑名單同樣支援萬用字元

### 顏色標記
- 每組規則可指定標記顏色和透明度
- 全域白名單也有獨立的標記顏色
- 匯出的 PDF 中，匹配的文字區域會以指定顏色覆蓋高亮

### Trial Run（試行模式）
- 預覽所有頁面的匹配結果，不實際匯出
- 結果表格包含：頁碼、包含/排除狀態、命中關鍵字、排除原因、文字預覽
- 快速調整規則後可重複 Trial Run，直到結果滿意

### 匯出 PDF
- 將所有符合條件的頁面複製至新 PDF
- 在頁面上繪製顏色高亮標記
- 自動下載結果檔案

## 使用流程

### 基本搜尋

1. **切換模組**：從左側 Activity Bar 點擊「🔍 PDF 搜尋」
2. **上傳 PDF**：點擊「📂 上傳 PDF」選擇檔案
3. **輸入關鍵字**：在第一組規則的 Keywords 欄位輸入搜尋關鍵字（逗號分隔多個）
4. **Trial Run**：點擊「▶ Trial Run」預覽匹配結果
5. **檢查結果**：在結果表格中確認哪些頁面被選中
6. **下載**：點擊「⬇ Download」匯出符合條件的頁面

### 進階搜尋（多規則 + 萬用字元）

1. **上傳 PDF** 並確認檔名顯示
2. **設定全域白名單**（可選）：輸入頁面必須包含的關鍵字，例如 `Bill No*`
3. **設定全域黑名單**（可選）：輸入要排除的關鍵字，例如 `DRAFT, SUPERSEDED`
4. **新增規則**：點擊「＋ 新增規則」建立多組搜尋條件
5. **使用萬用字元**：例如輸入 `Section ?.?` 匹配 `Section 1.1`、`Section 2.3` 等
6. **設定排除名單**：在每組規則的 Exclude 欄位輸入該規則的排除關鍵字
7. **選擇顏色**：為每組規則設定不同的標記顏色和透明度
8. **Trial Run**：預覽結果並調整規則
9. **Download**：確認後下載匯出 PDF

### 實際應用範例

#### 範例一：從 BQ 文件中提取特定 Bill
- 白名單：`Bill No. 3`
- 黑名單：`Appendix`
- 結果：只匯出包含 "Bill No. 3" 且不含 "Appendix" 的頁面

#### 範例二：提取所有包含價格的頁面
- 規則 1 Keywords：`$*`（使用萬用字元匹配任何金額）
- 規則 1 Exclude：`N/A, TBC`
- 結果：匯出含金額且未標記 N/A 或 TBC 的頁面

#### 範例三：按修訂版本過濾
- 規則 1 Keywords：`Rev ?, Revision ?`（匹配單一字元版本號）
- 白名單：`Structural`
- 結果：只匯出結構相關且標明版本號的頁面

### 小技巧

- 先用 Trial Run 測試規則效果，避免下載不需要的頁面
- 萬用字元 `*` 可放在任何位置，例如 `*concrete*` 匹配所有含 "concrete" 的文字
- 多組規則之間是 OR 關係 — 任一規則匹配即可選中頁面
- 排除名單只影響其所屬的規則，不影響其他規則的匹配
- 關鍵字搜尋不區分大小寫

## 限制

- 文字提取依賴 PDF 內嵌的文字層；掃描影像型 PDF 需先 OCR 才能搜尋
- 大型 PDF（數百頁以上）在瀏覽器內處理可能需要較長時間
- 高亮標記是以矩形覆蓋方式繪製，位置基於文字座標，偶爾可能略有偏移
