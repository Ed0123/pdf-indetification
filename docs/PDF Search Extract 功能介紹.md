# PDF 搜尋與擷取 功能介紹

## 概述

「PDF 搜尋與擷取」模組讓使用者針對多頁 PDF 檔案，透過自訂關鍵字規則搜尋頁面內容，
並將符合條件的頁面擷取到新的 PDF 檔案中下載。所有處理皆在瀏覽器本機完成。

## 功能

### 關鍵字規則管理
- 可新增多組搜尋規則，每組規則包含關鍵字、顏色標記
- 支援「全域白名單」模式：只要該頁包含任一白名單關鍵字即納入結果
- 支援「全域黑名單」模式：同時包含黑名單關鍵字的頁面不納入
- 每條規則可設定自身的排除關鍵字清單

### Trial Run（試行模式）
- 先提取所有頁面文字，快速預覽每頁是否匹配規則
- 顯示結果表格含頁碼、匹配/排除狀態
- 幫助使用者在正式匯出前調整規則

### 匯出 PDF
- 將符合規則的頁面複製至新 PDF
- 自動下載結果檔案

## 技術細節

| 項目 | 說明 |
|------|------|
| 處理位置 | 純前端（瀏覽器端） |
| 文字提取 | pdfjs-dist（Mozilla pdf.js） |
| PDF 組裝 | pdf-lib |
| 權限控制 | 透過 `pdf_search` feature flag 控制 |

## 使用流程

1. 從左側 Activity Bar 選擇「🔍 PDF 搜尋」模組
2. 上傳 PDF 檔案
3. 新增搜尋規則（關鍵字、顏色）
4. 設定全域白名單/黑名單（可選）
5. 點擊「Trial Run」預覽匹配結果
6. 確認結果後點擊「Download」匯出符合條件的頁面

## 限制

- 文字提取依賴 PDF 內嵌的文字層；掃描影像型 PDF 需先 OCR 才能搜尋
- 大型 PDF（數百頁以上）在瀏覽器內處理可能需要較長時間
- 關鍵字搜尋為精確字串比對（不支援正則表達式）
