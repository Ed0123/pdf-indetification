# Excel Export 功能介紹

## 概述

「Excel Export」模組提供類似 Excel 的表格介面，一覽所有已載入 PDF 頁面的擷取資料。使用者可以搜尋篩選、排序、逐格編輯，最終匯出為 Excel（.xlsx）檔案下載。

## 功能

### 資料瀏覽
- 以表格呈現所有頁面的擷取結果，包含檔名、頁碼、已套用模板、以及各自訂欄位
- 標題列統計：總頁數、含資料的頁數、已套用模板的頁數

### 搜尋與篩選
- 即時搜尋框，可依檔名、文字內容或模板名稱過濾
- 篩選結果立即更新表格

### 欄位控制
- 「Columns ▾」按鈕開啟欄位選單，勾選/取消勾選以顯示或隱藏特定欄位
- 點擊表頭排序（▲ ▼ 切換升降序）

### 逐格編輯
- 雙擊任何儲存格進入編輯模式
- 修改完按 Enter 儲存並跳到下一行
- 按 Escape 取消編輯

### 匯出 Excel
- 點擊「📥 Export to Excel」將所有篩選後的資料下載為 .xlsx 檔案

## 使用流程

1. **確認資料來源**：先在「PDF Pages」模組上傳 PDF 並擷取/輸入資料
2. **切換模組**：從左側 Activity Bar 點擊「📊 Excel Export」
3. **瀏覽資料**：所有頁面的資料自動呈現在表格中
4. **搜尋與篩選**：在搜尋框輸入關鍵字，快速定位目標頁面
5. **校對與編輯**：雙擊儲存格修正資料內容
6. **隱藏不需要的欄位**：點擊「Columns ▾」取消勾選不需匯出的欄位
7. **排序**（可選）：點擊表頭按需排序
8. **匯出**：點擊「📥 Export to Excel」下載 .xlsx

### 快捷鍵

| 操作 | 快捷鍵 |
|------|--------|
| 進入編輯 | 雙擊儲存格 |
| 儲存並跳下一行 | Enter |
| 取消編輯 | Escape |

### 小技巧

- 匯出前先用搜尋篩選出目標範圍，只匯出需要的頁面
- 排序後匯出，可控制 Excel 中的行順序
- 狀態列會顯示目前顯示的行數佔總行數的比例
