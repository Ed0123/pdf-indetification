# Single Page Mode

When toggled in the toolbar, Single Page Mode restricts the data table to show
only rows related to the current PDF page. Navigation buttons (Previous / Next)
and a page number indicator appear in the toolbar. All editing and template
behaviour remain identical to normal mode, and changes persist when switching
pages. Sorting and column adjustments are temporarily disabled during updates to
prevent data loss.

The feature is described in detail in the Specification and Architecture
documents; refer there for the latest behaviour and test scenarios.

*End of Single Page Mode overview.*
