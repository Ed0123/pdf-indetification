## Technical Details

### 1. BQ Classification Logic (backend/routers/bq.py)

```python
# Classification: item if has item_no AND (qty OR unit)
row_type = "item" if (item_no and (quantity is not None or unit)) else "notes"
```

Uses y-coordinate alignment (tolerance 5px) to group words into rows, then assigns words to columns based on x-position within column bounding boxes.

### 2. JSON Export Format

```json
{
  "id": 1,
  "project_id": "T26008",
  "Type": "Item",
  "bill": "S3",
  "page": "3",
  "item": "B",
  "revision": "TA0",
  "ref": "S3/3/B",
  "data_detail": "description text...",
  "qty": "50",
  "unit": "Item",
  "rate": "50",
  "total": "2500"
}
```

- `bill` and `page` parsed from `page_label` (format: "AA/BB")
- `ref` = `bill/page/item`
- `total` = `qty * rate` (calculated if both present)

### 3. Resizable Panels

- Vertical resize: Mouse drag on horizontal handle between DataTable and SinglePageDataTable
- Horizontal resize: Mouse drag on vertical handle between content and PDF viewer
- State stored in `dataTableHeight` (%) and `contentWidth` (%)
