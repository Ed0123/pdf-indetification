# 網站功能介紹

> **Note (2026-06-17):** 更新功能列表、UI 改進說明。

此文件簡要說明 QS 小工具（Tools for QS）在瀏覽器中可使用的主要功能，
讓使用者快速了解我們的工具集合。

## 核心工具

- **BQ-OCR**
  - 從工程量表 PDF 中自動擷取文字內容，
  - 支援欄位識別、頁面小計、格式化輸出。
- **BQ 匯出**
  - BQ 數據預覽、下載、頁面瀏覽。
- **PDF OCR 與文件處理**
  - 上傳任意 PDF，框選範圍進行文字辨識，
  - 支援匯出為 Excel、JSON、PDF 等格式。
- **PDF & Excel 解鎖**
  - 瀏覽器本機解鎖 PDF 限制、Excel 工作表保護。
  - Excel 解鎖完整保留圖片、公式、繪圖等內容。
- **PDF 搜尋與擷取**
  - 關鍵字搜尋 PDF 頁面，支援萬用字元（`*`、`?`）。
  - 擷取符合條件的頁面下載。

## 使用情境

1. 登入後預設進入「主頁」首頁，可快速查看帳號、配額、功能權限。
2. 從左側 Activity Bar 切換至各功能模組。
3. 各模組提供「📖 說明」按鈕，點擊彈出說明視窗（管理員可即時編輯）。
4. 編輯完成後匯出所需格式，或將資料同步至雲端。

## 功能權限顯示

主頁「功能權限」卡片將功能分為三類：
- **已開啟**：一般開關型功能（如匯出 Excel、PDF 解鎖等）
- **未開啟**：尚未啟用的功能
- **限額類**：涉及用量配額的功能（如 OCR、雲端儲存、自動備份）

## 管理員面板

管理員面板分為 3 個子頁籤：
1. **使用者列表**：搜尋、排序、篩選用戶，可直接修改類別、群組、狀態、備註
2. **群組與配額**：群組管理、會員類別/配額管理、功能矩陣
3. **使用者訊息**：查看及回覆用戶訊息

## 其他特色

- 使用者帳號採用 Google 登入，資料可安全同步。
- 本機優先儲存，雲端備份確保最低成本。
- 系統設計以測量師（QS）為主要受眾，操作流程簡潔、直覺。
- 獨立模組（主頁、PDF/Excel 解鎖、PDF 搜尋）隱藏左側 PDF 樹和右側 PDF 預覽，版面更簡潔。
