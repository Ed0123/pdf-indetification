# BQ Export 功能介紹

此文檔為 BQ Export 面板的使用說明。包含常見工作流程、鍵盤快捷鍵、以及各項功能的快速導覽。

## 概覽
- 查看和編輯從 PDF 中透過 BQ OCR 提取的資料。
- 支援各種格式匯出（Excel、JSON、PDF 等）。

## 核心功能

1. **鍵盤導覽**
   - 使用箭頭鍵移動；Enter/F2 進入編輯。
   - Insert 新增行。
   - 跳過 `Notes` 行以提高效率。
2. **頁面小計**
   - 每頁最後一行顯示該頁總計，數值在 `Subtotal` 欄位。
3. **欄位操作**
   - 拖動欄位邊界調整寬度。
   - 按上方的 × 隱藏欄位，點擊「取消隱藏」還原。
   - 在標題旁切換 `Item` / `Ref` 標籤。
4. **說明視窗**
   - 按下右上角的 ❓ 按鈕顯示快速說明。

## 其他提示
- 資料變更會自動同步至資料庫。
- 若需重新計算所有價格，點擊「🔄 重新計算」。
