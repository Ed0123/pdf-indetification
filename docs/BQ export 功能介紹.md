# BQ Export（工程量表匯出）功能介紹

## 概述

「BQ Export」模組提供類似 Excel 的進階表格介面，讓使用者檢視、編輯、校對所有透過 BQ OCR 辨識的工程量表資料，並支援多種匯出格式：JSON、Excel (CSV)、PDF（含或不含標註）。功能包括鍵盤導覽、多格選取、複製貼上、自動計算合計、以及 Collection 頁面交叉參照。

## 功能

### 資料瀏覽與篩選
- **類型篩選**：依行類型篩選 — All / Items / Notes / Collection Pages / Collection Entries
- **欄位篩選**：每個欄位下方都有搜尋輸入框，可即時篩選
- **排序**：點擊表頭切換升序/降序/取消排序
- **隱藏/顯示欄位**：每個表頭右上角有 ✕ 按鈕可隱藏該欄；隱藏後可點擊「👁 顯示隱藏欄」一鍵恢復
- **拖曳調整欄寬**：拖曳表頭右邊框可調整欄位寬度，如同 Excel 操作
- **頁面小計行**：每頁資料末端自動顯示頁面小計行（綠色底 = 普通頁 / 藍色底 = Collection 頁）

### 鍵盤導覽（核心功能）
- 使用箭頭鍵在儲存格之間移動，類似 Excel 體驗
- 導覽時自動跳過 Notes 行，提高效率
- Tab / Shift+Tab 左右移動

### 逐格編輯
- 按 Enter 或 F2 開始編輯
- 直接打字即可開始輸入
- 數值欄位自動驗證格式
- 編輯後的儲存格以藍色粗體顯示，方便辨識修改過的資料

### 自動計算
- 輸入 Qty（數量）和 Rate（單價）後，Total（合計）自動計算
- 「🔄 重新計算」按鈕：一次重新計算所有行的合計
- 每頁最後一行自動顯示該頁小計

### 多格選取 & 複製貼上
- Shift + 箭頭鍵擴展選取範圍
- Ctrl+C 複製選取範圍（Tab 分隔格式）
- Ctrl+V 從 Excel 或其他表格貼上資料
- Delete / Backspace 清除選取儲存格

### 行操作
- Insert 鍵或 ➕ 按鈕在當前行下方插入新行
- 🗑️ 按鈕刪除整行
- 行類型下拉選單可切換 item / notes / collection

### Collection 交叉參照
- Collection entry 行可透過下拉選單選擇參照頁面
- 「重新計算」會自動將各頁小計填入 Collection entry 的 Total

### 新增欄位
- **Trade**：工種分類（預設隱藏，可顯示）
- **Group**：群組標籤（預設隱藏，可顯示）
- **Remark**：備註欄位，位於 Total 與 Action 之間

### 多格式匯出
- **JSON**：完整資料含標註及元資料
- **Excel (CSV)**：標準 Excel 格式
- **XLSX (Data)**：標準 Excel .xlsx 格式，含 Trade/Group/Remark 欄位
- **XLSX (Summary)**：按頁彙總摘要 .xlsx 格式
- **Page Totals**：按頁彙總摘要 CSV
- **PDF (Original)**：原始頁面匯出
- **PDF (with User Input)**：頁面上疊加使用者編輯的值

匯出前可依頁面、版本號或行類型篩選匯出範圍，並可自訂 Project ID。

## 使用流程

### 校對資料

1. **切換模組**：從左側 Activity Bar 點擊「📊 BQ Export」
2. **瀏覽資料**：BQ OCR 辨識的結果自動顯示在表格中
3. **篩選類型**：從類型篩選下拉選單選擇要查看的行類型
4. **使用欄位篩選**：在 Item、Description 等欄位下方的搜尋框輸入關鍵字定位
5. **編輯修正**：點擊儲存格或按 Enter 進入編輯，修正 OCR 辨識錯誤
6. **重新計算**：點擊「🔄 重新計算」更新所有 Total

### 匯出資料

7. **點擊「📥 Export」**：開啟匯出設定面板
8. **輸入 Project ID**（可選）：預設為 `BQ_{日期}` 格式
9. **設定篩選條件**（可選）：依頁面、版本或類型限制匯出範圍
10. **選擇匯出格式**：點擊對應的匯出按鈕下載

### 快捷鍵

| 操作 | 快捷鍵 |
|------|--------|
| 移動焦點 | ⬆ ⬇ ⬅ ➡ |
| 進入編輯 | Enter / F2 |
| 下一欄 | Tab |
| 上一欄 | Shift + Tab |
| 擴展選取 | Shift + 箭頭鍵 |
| 複製 | Ctrl + C |
| 貼上 | Ctrl + V |
| 清除 | Delete / Backspace |
| 插入新行 | Insert |
| 取消編輯 | Escape |

### 小技巧

- 從 Excel 複製多行資料後，在表格中 Ctrl+V 可批量貼入
- 利用「重新計算」一次修正所有自動合計，避免手動逐行檢查
- 匯出 JSON 適合後續程式處理；匯出 PDF (with User Input) 適合存檔備查
- 頁面小計行無法直接編輯，會由系統自動計算
- 先篩選再匯出，只匯出需要的章節或範圍
