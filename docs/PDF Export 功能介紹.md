# PDF Export 功能介紹

## 概述

「PDF Export」模組讓使用者從已載入的多個 PDF 中，勾選需要的頁面並自訂每頁的匯出檔名，最終打包為 ZIP 壓縮檔下載。適合需要從大量 PDF 中提取特定頁面、並依自訂命名規則整理的工作場景。

## 功能

### 頁面選擇
- 以勾選框逐頁選擇要匯出的頁面
- 「全選/取消全選」按鈕批量操作
- 支援篩選後再全選，只選中顯示中的頁面

### 篩選
- 搜尋框可依檔名或自訂標籤即時篩選
- 篩選會即時更新表格

### 自訂匯出檔名
- 每頁預設檔名格式為 `{原始檔名}_P{頁碼}`
- 雙擊「Export Label」欄位可輸入自訂名稱
- 自訂名稱將作為匯出 PDF 的檔名

### 匯出
- 點擊「📥 Export {N} PDF(s) as ZIP」下載 ZIP 壓縮檔
- ZIP 內每個檔案為獨立的單頁 PDF，以自訂或預設名稱命名

## 使用流程

1. **確認資料來源**：先在工具列上傳需要的 PDF 檔案
2. **切換模組**：從左側 Activity Bar 點擊「📄 PDF Export」
3. **篩選**（可選）：在搜尋框輸入關鍵字，縮小可見範圍
4. **勾選頁面**：逐頁勾選，或點擊「全選」一次選取所有可見頁面
5. **自訂檔名**（可選）：雙擊 Export Label 欄位，輸入想要的檔案名稱，按 Enter 確認
6. **匯出**：點擊底部的匯出按鈕，瀏覽器自動下載包含所選頁面的 ZIP 檔

### 快捷鍵

| 操作 | 快捷鍵 |
|------|--------|
| 編輯匯出標籤 | 雙擊 Export Label 欄位 |
| 確認標籤 | Enter |
| 取消編輯 | Escape |

### 小技巧

- 利用篩選框配合全選，快速選取特定檔案的所有頁面
- 先在 Export Label 中統一命名格式，方便後續檔案管理
- 匯出後 ZIP 內的檔案可直接解壓使用，無需再手動拆分 PDF
